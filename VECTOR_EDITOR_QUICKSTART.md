# 向量编辑器快速入门

## 快速开始（3步骤）

### 步骤 1: 加载向量
```
打开 "Vector Editor" 区域
↓
点击 "From Preset" 或 "From Object"
↓
选择要编码的几何体
```

### 步骤 2: 编辑向量
```
方式A: 直接编辑32个数值参数
方式B: 使用向量运算（加减乘除、插值）
```

### 步骤 3: 渲染结果
```
点击 "Decode & Render"
↓
在视口中查看3D对象
```

## 常用操作

### 🎯 混合两个预设（最常用）
1. Load Preset A → **Copy**
2. Load Preset B
3. Operation: **Interpolate**, Factor: **0.5**
4. **Apply** → **Decode & Render**

### ✏️ 手动微调
1. Load Preset
2. 调整特定参数（如 [2-4] 改变尺寸）
3. **Decode & Render**

### 🔄 创建变体
1. Load Preset → **Copy**
2. Load Same Preset
3. 修改几个参数
4. Operation: **Add**
5. **Apply** → **Decode & Render**

### 📏 缩放几何体
1. Load Vector
2. Operation: **Multiply**, Scalar: **2.0**
3. **Apply** → **Decode & Render**

## 向量运算公式

### Add (加法)
```python
result = current + clipboard
用途: 组合两个几何特征
```

### Subtract (减法)
```python
result = current - clipboard
用途: 提取差异特征
```

### Multiply (乘法)
```python
result = current * scalar
用途: 缩放所有特征
```

### Interpolate (插值)
```python
result = (1-t) * current + t * clipboard
用途: 平滑过渡，t=0是current，t=1是clipboard
```

## 小技巧 💡

### Tip 1: 使用剪贴板作为备份
在进行大量修改前，先 **Copy** 当前向量。如果结果不满意，可以 **Paste** 恢复。

### Tip 2: 渐进式调整
不要一次性修改太多参数。每次修改1-2个参数后立即 **Decode** 查看效果。

### Tip 3: 归一化技巧
当向量值变得很大时，使用 **Normalize** 可以将其缩放到单位长度，便于控制。

### Tip 4: 插值探索
使用 Interpolate 在两个已知好的设计之间探索，Factor从0到1慢慢调整。

### Tip 5: 保存中间结果
通过 **Copy** 保存有趣的中间状态到剪贴板，可以随时返回。

## 常见问题

**Q: 为什么Decode后看不到对象？**  
A: 确保场景中有基础几何体，或者使用 "Update Shape" 创建新对象。

**Q: 可以撤销向量修改吗？**  
A: 使用 Copy/Paste 手动管理"撤销"，或在修改前保存场景文件。

**Q: 哪些参数最重要？**  
A: [2-4] (尺寸), [12-13] (扭曲/锥度), [19-21] (形状变形) 通常效果最明显。

**Q: 如何创建动画？**  
A: 使用 "Morph Animation" 操作符创建插值动画序列。
